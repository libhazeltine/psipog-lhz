First off, get some good hosting.

Secondly, you need to have a goal for your site, something that makes it stand out. If you do a site about "everything" you're basically just mimicing psipog, and not offering anything new.

Also, take it from me, as I have experiemce with this. .tk domains suck.

Note, please dont use frontpage. It adds lots of XML elements you dont actually need, it also is VERY sloppy. If you want to make a good website, learn to code for yourself. Get away from doing traditional HTML and use XHTML 1.0 Strict, and for god's sake, learn CSS, it will make everything run faster and easier.

Just some advice comming from someone who had to learn the hard way.
