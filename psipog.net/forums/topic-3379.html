The definition that PsiPog uses is...

Something is religious if it indirectly implies, or directly answers one or both of the following questions: 1. What happens when we die? 2. How were we created?

Talking about ghosts almost always implies an afterlife, which would be a partial answer to #1, which is considered religious.

For more information on why this rule exists, and why we use this definition of religion, please see http://psipog.net/forums/viewtopic.php?t=3036
