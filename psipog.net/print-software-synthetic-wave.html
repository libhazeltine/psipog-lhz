Software: Synthetic Wave for Meditation

by Peebrain

I've written a simple program to help in meditation. It uses pink noise to randomly simulate ocean sounds. It also includes an alarm function, so that if you want to end your meditation after 20 minutes (for example), then it will fade into an alarm when the time is up.

If you download and use this software, you are doing so at your own risk. I'm releasing it under the GNU General Public License. It was created and compiled with Microsoft Visual C++ 6.0.

If you notice any bugs, or have suggestions, please contact me at peebrain@psipog.net. It should be pretty straight forward to use. Have fun!

- Sean (aka Peebrain)
