Musical scales come from logarithms. Our 12 note system is calculated using the 12th root of 2, which is about 1.059463. If you'd like to make (for example), a 7 note system, you would use the 7th root of 2 (1.1040895..). The reason for this is because of the concept of octaves, where a note's frequency is doubled when moving up an octave.

So for a guitar, the formula is:

fret distance = (first fret distance) * (12th root of 2)^(fret number)
