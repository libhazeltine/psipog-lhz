July 14th, 2008 by SeanLottery Analysis

I’ve analyzed my results from the past 17 months, concerning the lottery.

I devised a way to measure how “close” I get to the winning numbers.  My formula for measuring this “closeness” is somewhat arbitrary, but there is no correct formula, so it works as good as any.  In calculating the odds, I measure the “closeness” of all possible tickets - then see how many tickets are just as “close” as my ticket.

For the curious programmers, you can see my C program for measuring “closeness” here: Lottery Score Calculator Source Code
