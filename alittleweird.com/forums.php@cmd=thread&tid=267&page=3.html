you dont run sbagen itself, you install it with the proper windows installer from their website, then whenever you run an SBagen file (.sbg) it will load itself into sbagen and run in a command window.

also,
your headphones might not be stereo? or you may have the sound output set to mono on your computer? (if there are additional adapters/wires between your headphones and the back/front of the computer one of them may be downmixing the signal from stereo to mono) - also double check the wire(s) are all plugged into their respective jacks all the way since a half plugged jack can sometimes provide non-stereo sound.
